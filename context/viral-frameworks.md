# Viral Frameworks — Cómo hacer contenido que se quede viendo

> Psicología de retención para Gen Z y Millennials.
> El algoritmo premia tiempo de visualización. La retención premia el contenido.

---

## FRAMEWORK 1: THE LOOP (el más viral en TikTok/Reels)

El final del video lleva directo de regreso al inicio. El cerebro lo ve de nuevo sin darse cuenta.

```
Estructura:
Apertura: [afirmación incompleta]
Desarrollo: [revela información gradualmente]
Final: [cierra con la misma imagen/frase del inicio → loop]

Ejemplo:
Inicio: "Esto transformó la cocina en un día" [muestra cocina renovada]
Medio: [proceso de transformación]
Final: [regresa a la misma cocina del inicio → el cerebro quiere ver de nuevo]
```

---

## FRAMEWORK 2: OPEN LOOP + PAYOFF

Crear tensión desde el inicio, resolver al final. El cerebro NO puede irse sin la respuesta.

```
0-3s:  Pregunta o afirmación que NO se puede ignorar
3-15s: Aumentar la tensión (datos, contexto, problema)
15-25s: Falso cierre (casi revelas pero no)
25-30s: PAYOFF — la respuesta / la transformación
```

---

## FRAMEWORK 3: BEFORE/AFTER CON EMOCIÓN

No mostrar solo el resultado — mostrar la EMOCIÓN del cambio.

```
1. Muestra el problema (con música baja, colores opacos)
2. Pausa dramática (silencio o sonido de suspense)
3. Reveal (cambio de música, colores brillantes, zoom out)
4. Reacción emocional (cara, texto, música eufórica)
```

---

## FRAMEWORK 4: EDUTAINMENT (educar + entretener)

El contenido más guardado y compartido. Enseña algo real en formato entretenido.

```
Hook: "Nadie te explica esto sobre [tema]"
Punto 1: Dato sorprendente
Punto 2: Por qué importa
Punto 3: Qué hacer al respecto
CTA: "Guarda esto para cuando lo necesites"
```

---

## FRAMEWORK 5: CONTRARIAN (ir contra la corriente)

Engagement altísimo. La gente comenta para defender o atacar. El algoritmo ama los comentarios.

```
"Todos dicen [consejo popular]. Está MAL."
↓
Explica por qué el consejo popular falla
↓
Da tu alternativa con evidencia/resultado
↓
CTA: "¿Estás de acuerdo?" (invita a discutir)
```

---

## FRAMEWORK 6: STORYTELLING EN 3 ACTOS

El cerebro humano está programado para historias. No se puede parar.

```
Acto 1 (10%): El personaje en su peor momento
Acto 2 (60%): El conflicto / el camino / el proceso
Acto 3 (30%): La resolución + lección
```

Para marcas:
```
"Nuestra empresa casi cierra en [año]. Aquí lo que pasó y cómo sobrevivimos."
```

---

## RETENCIÓN POR TIEMPO (dónde la gente se va)

```
0-1.5s   → Si no hay hook, 60% se va aquí
1.5-3s   → Si el visual no promete algo bueno, 30% más se va
3-7s     → Drop importante — necesitas un "micro-hook" aquí también
15s      → Otro drop — introduce nueva información o giro
Último 5s → Los que llegan aquí son fans → CTA fuerte aquí
```

**Táctica:** Poner micro-hooks cada 3-5 segundos. Texto, zoom, sonido, dato nuevo.

## SONIDOS Y MÚSICA (impacto en retención)

| Tipo | Efecto |
|------|--------|
| Drop de música en el reveal | +40% retención en el momento |
| Silencio súbito | Para el scroll más que cualquier visual |
| Sonido de notificación | Condicionamiento pavloviano — activa atención |
| Voz en off + subtítulos | +35% vs solo subtítulos |
| ASMR / sonido satisfactorio | Shares orgánicos altísimos |
| Trending audio | Boost algorítmico de TikTok/IG |

---

## COLORES Y CONTRASTE (psicología visual)

| Color | Emoción | Usar cuando |
|-------|---------|-------------|
| Verde lima (#d2e823) | Energía, acción, novedad | CTAs, revelaciones |
| Azul marino oscuro | Confianza, premium, estabilidad | Marca, credibilidad |
| Rojo/naranja | Urgencia, peligro, emoción | FOMO, problemas, alertas |
| Blanco sobre negro | Claridad, impacto, lujo | Headlines fuertes |
| Beige/crema | Calidez, naturali, artesanal | Marcas lifestyle |

**Contraste:** texto y fondo SIEMPRE con ratio mínimo 4.5:1. Si no se lee en 0.5s, no sirve.

---

## CAPTIONS / SUBTÍTULOS

- **85%** de videos en redes se ven sin audio
- Subtítulos aumentan watch time en promedio **12%**
- Estilo TikTok: texto grande, centrado, palabra por palabra
- Color del texto: blanco con sombra negra O negro con fondo blanco
- Nunca más de 5 palabras por frame

---

## MÉTRICAS QUE IMPORTAN (en orden)

1. **Watch time %** — cuánto del video ven (lo más importante)
2. **Saves** — señal de valor real
3. **Shares** — señal de contenido viral
4. **Comments** — señal de engagement
5. **Likes** — la menos importante pero la más visible

---

## VIRALITY CHECKLIST — Antes de publicar cualquier pieza

- [ ] ¿El primer 1.5s tiene hook visual + texto?
- [ ] ¿El texto se lee en 0.5 segundos?
- [ ] ¿Hay contraste suficiente entre texto y fondo?
- [ ] ¿Tiene subtítulos o texto en pantalla?
- [ ] ¿El ritmo es rápido (corte cada 2-3s en video)?
- [ ] ¿Hay un loop abierto o pregunta sin responder?
- [ ] ¿El CTA es específico ("comenta X", "guarda esto", "envíame DM")?
- [ ] ¿La música/sonido coincide con la emoción del contenido?
- [ ] ¿Podría funcionar sin audio?
- [ ] ¿Un desconocido entiende el valor en 3 segundos?
